Le duchamp_reader est une application qui permet de découvrir les dernières éditions du prix Marcel Duchamp (2018-2021). Créé en 2000 en partenariat avec l'ADIAF (Association pour la diffusion internationale de l'art français) pour mettre en lumière la scène artistique française, le prix Marcel Duchamp cherche à rendre visible la diversité des pratiques artistiques contemporaines. Chaque année, quatre artistes sont nominé.e.s. Un.e lauréat.e est ensuite désigné.e après le vote d'un jury.
Le duchamp_reader s'appuie sur les librairies python flask et flask_sqlalchemy; il a été développé dans le cadre du master 2 Techniques numériques appliquées à l'Histoire de l'École nationale des chartes

Le site est divisé en parties :

Artistes
les artistes nominé.e.s au prix Marcel Duchamp
Nomination
la nomination de chaque artiste au prix Marcel Duchamp
Galeries
les galeries représentant les nominé.es
Villes
les villes d'origine et de résidence des artistes
Thèmes
les thèmes sur lesquels portent les nominations au prix Marcel Duchamp
Le prix Marcel Duchamp en statistiques
découvrir des statistiques sur le prix Marcel Duchamp

Le code source du site est disponible sur Github
